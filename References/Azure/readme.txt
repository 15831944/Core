The project contains client libraries you can add to your project to consume 
Azure services.

It is recommended that you grab the latest client libraries using NuGet instead
of directly adding them from this folder.

Instructions for obtaining the latest client libraries are available at the
following locations:
- Storage: http://azure.microsoft.com/en-us/documentation/articles/storage-dotnet-how-to-use-blobs-20/#configure-access
- Service Bus: http://azure.microsoft.com/en-us/documentation/articles/service-bus-dotnet-how-to-use-queues/#span-classshort-headerget-the-nuget-packagespanget-the-service-bus-nuget-package
- All: http://www.nuget.org/packages?q=windowsazureofficial
